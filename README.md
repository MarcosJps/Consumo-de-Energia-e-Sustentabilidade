# ⚡ Análise de Consumo de Energia e Sustentabilidade

Este repositório contém o projeto de clusterização de dados de consumo de energia elétrica, desenvolvido para a disciplina de Inteligência Artificial.

## 📌 Sobre o Projeto
O objetivo deste projeto é aplicar algoritmos de aprendizado não supervisionado (**K-Means** e **DBSCAN**) para agrupar e identificar padrões no consumo de energia doméstica. 
A base de dados utilizada é a **Household Electric Power Consumption**.

## 🚀 Como executar o projeto no Google Colab

Para testar as análises e gerar os gráficos corretamente, siga o passo a passo abaixo:

**1. Acesse o Notebook:**
* Faça o download do arquivo `.ipynb` deste repositório.
* Abra o [Google Colab](https://colab.research.google.com/) e faça o upload do arquivo para o seu ambiente.

**2. Prepare a Base de Dados:**
* Faça o download da base de dados disponibilizada neste repositório. (Se você subiu em `.zip`, lembre-se de extrair para obter o `.txt` original).
* No Google Colab, com o notebook aberto, clique no ícone de **Pasta** (Arquivos) na barra lateral esquerda.
* Clique no botão de **Upload** e selecione o arquivo `household_power_consumption.txt`. 
* *Nota: o arquivo ficará no armazenamento temporário da sessão do Colab.*

**3. Rode o Código:**
* Vá no menu superior do Colab, clique em **Ambiente de execução** e depois em **Executar tudo** (ou use o atalho `Ctrl + F9`).
* Aguarde o processamento dos dados e visualize os gráficos de clusterização!
